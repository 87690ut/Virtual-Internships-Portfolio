# Virtual Internships Portfolio

## 📌 Overview
This repository serves as a centralized portfolio for real-world data analytics case studies completed through various virtual internship programs, such as Forage. The primary focus of these projects is data cleaning, exploratory data analysis (EDA), feature engineering, and translating raw datasets into actionable business strategies.

## 🚀 Current Projects

### 1. Retail Data Analysis & Customer Segmentation
* **Objective:** Analyzed transaction datasets and customer demographics to identify purchasing behaviors, determine key drivers of sales, and provide commercial recommendations for a Category Manager.
* **Tech Stack:** Python, Jupyter Notebook, Pandas, Matplotlib.
* **Key Tasks Performed:**
  * **Data Quality Assessment:** Identified and handled missing values, inconsistencies, and outliers in transaction records.
  * **Feature Engineering:** Extracted valuable metrics such as 'pack size' and 'brand name' from product descriptions.
  * **Data Merging:** Integrated customer data with transaction data for a comprehensive view.
  * **Strategic Insights:** Defined customer segments and created visual reports to guide future targeting strategies.
